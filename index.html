Anonymous Nostr Note Sender

Anonostr allows users to send anonymous notes to the Nostr network without revealing their identity. The app generates a new key pair for each note submission and publishes the note using a few select relays. It supports tagging, threading, and replying to existing notes.

ℹ️ Enable this to link your notes as replies in consecutive order within a threaded conversation.
